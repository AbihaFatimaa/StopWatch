⏱️ STOPWATCH WEB APP

A modern, responsive stopwatch built using HTML, CSS, and JavaScript with smooth UI, dark mode, and keyboard controls.


🚀 Live Demo

👉 https://abihafatimaa.github.io/StopWatch/

✨ Features

▶️ Start / Stop toggle

🔄 Reset stopwatch

🌙 Dark mode / Light mode

⌨️ Keyboard shortcuts

Space → Start/Stop
R → Reset
📱 Fully responsive design (mobile-friendly)

⏱️ Accurate time tracking using JavaScript

🎨 Clean modern UI with smooth hover effects


🛠️ Tech Stack

HTML5

CSS3 (Flexbox, Media Queries, Glassmorphism)

JavaScript (DOM, Events, setInterval)
